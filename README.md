This is a simple URL Shortener web application built using Flask and SQLite.

## Features
- Shortens long URLs
- Stores original and short URLs in database
- History page to view saved URLs

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite
- SQLAlchemy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
